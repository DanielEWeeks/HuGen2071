5.4.1 Question 1

The clinical group, which measured height, used integer IDs, but the medical group, which measured the blood pressure, decided to prefix the integer IDs with the string ‚ÄòSG‚Äô (so as to distinguish them from other studies that were also using integer IDs). So ID ‚Äò1‚Äô was mapped to ID ‚ÄòSG0001‚Äô.

The clin data frame
ID height
1 152
104 172
2112 180
2543 163

Discuss how, using R commands, you would reformat the integer IDs to be in the format ‚ÄúSGXXXX‚Äù. Write down your ideas in the next section, and, if you have time, try them out within an R chunk.

Hint: Use the formatC function.
